Battlecode links

Here are some links to Battlecode information. I read a lot of these to motivate myself for the 2014 tournament!

  • Cory Li's intro to Battlecode
  • Cory Li on bytecode optimization for Battlecode 
  • A post on Battlecode by Greg Little, infamous for some of the hacks he describes there
  • 2009 winners gtg ice skating lessons' strategy report
  • Steve Arcangeli's post-mortem as a member of the 2011 winning team gunface-- see also his code snippets
  • 2012 winners fun gamers' strategy report -- see also their code repository 
  • Ryan Liu's post-mortem as a member of the 2013 winning team Teh Nubs -- see also their code repository
  • A nice post on Battlecode 2014 by David Westreicher of team schnitzel
  • My recollections as 2014 winner and my suggestions for how to win -- see also my code repository which contains links to other 2014 repositories
  • The OpenCourseWare site for Battlecode 2013, including the game software and recorded lectures.
  • The Battlecode 2014 game software; see also the instructions on how to install it and compile a bot and the documentation for the API.
